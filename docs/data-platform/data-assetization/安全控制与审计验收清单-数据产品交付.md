# 安全控制与审计验收清单（数据产品交付形态）

> **文档编号**：COM-05
> **适用交易形态**：数据产品交付（文件/数据库导入）
> **版本**：v1.0
> **最后更新**：2026-01-24
> **用途**：可直接作为服务合同附件，用于验收和审计

---

## 使用说明

1. **验收时机**：数据产品交付前、交付时、交付后（持续监控）
2. **填写要求**：
   - "实施状态"必须为"已完成"才能通过验收
   - "证据材料"必须提供具体文件名或截图
   - "验收人"必须签字确认
3. **适用范围**：
   - **数据产品交付**：主要关注静态数据的访问控制、加密存储、审计记录
   - **API/数据服务**：需额外关注API限流、熔断、在线审计（参见附录A）
   - **联合计算**：需额外关注可信执行环境、多方计算审计（参见附录B）
