# 交易所红线自查表（数据产品交付形态）

> **文档编号**：COM-CHECKLIST-01
> **适用交易形态**：数据产品交付（文件/数据库导入）
> **版本**：v1.0
> **最后更新**：2026-01-24

---

## 使用说明

1. **自查时机**：数据资产登记准备阶段，在提交交易所前完成
2. **填写要求**：
   - 所有勾选项必须有负责人签字
   - "证据材料"栏必须填写具体文件名或路径
   - "风险评估"栏如为"是"，必须附整改方案
3. **审核流程**：技术团队填写 → 法务/合规复核 → 交易所审查
4. **留存要求**：原件随交易所登记材料提交，复印件留存3年以上
